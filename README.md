## 📋 About This Project

A professional registration form built with pure HTML, CSS and JavaScript.
This project focuses on **Data Integrity** and **User Experience** through
client-side form validation — no libraries, no frameworks, just clean vanilla code.

---

## ✅ Features

- 👤 Full Name validation (empty check)
- ✉️ Email format validation using Regex
- 🔒 Strong password enforcement (min 8 chars, uppercase, number, symbol)
- 🔑 Confirm password match check
- 🔴 Real-time red error messages per field
- 🟢 Green success message on valid submission
- 🚫 `event.preventDefault()` — page never refreshes
- 🎨 Animated dark background with floating frontend keywords
- 💜 Glassmorphism card design
